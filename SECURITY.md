# Security Policy

## Reporting a Vulnerability

Use GitHub's private security advisory mechanism as the primary channel:
- Navigate to the Security tab of this repository and select "Report a vulnerability".

For urgent or out-of-band contact, reach the maintainer at benjaminblake94@gmail.com.

This file is the CD.20 projection (T2.12 public-repo security gate). T2.11b may enrich it
with formal disclosure timelines and response SLAs.
