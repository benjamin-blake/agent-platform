# Security Policy

This file is a projection of the repository's security posture: vulnerability reporting is
enabled in GitHub repo settings and the security controls live as code in `terraform/github/`
(CD.20 public-repo security gate, T2.12) -- see `terraform/github/` and `CLAUDE.md` for
canonical content.

## Reporting a Vulnerability

Use GitHub's private security advisory mechanism as the primary channel:
- Navigate to the Security tab of this repository and select "Report a vulnerability".

For urgent or out-of-band contact, reach the maintainer at benjaminblake94@gmail.com.
